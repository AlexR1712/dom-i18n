dom-i18n is a lightweight (less than 1KB minified/gzipped) JavaScript module that offers an alternative for supporting internationalization on HTML static pages that have no better option than to serve many languages at once. dom-i18n est un module JavaScript très léger qui vous offre une alternative pour traduire facilement vos pages HTML. Très utile si vous avez une limitation au niveau du serveur qui vous empêche de servir des pages spécifiques pour chaque langue. dom-i18n é um módulo JavaScript que oferece uma alternativa para o suporte a internacionalização de páginas HTML. Bastante útil quando existem limitações no servidor que impedem a existência de diferentes versões da página para cada língua.
