dom-18n

dom-i18n is a lightweight (less than 1KB minified/gzipped) JavaScript library that offers an alternative for supporting internationalization on HTML static pages that have no better option than to serve many languages at once. dom-i18n est une simple bibliothèque en JavaScript pour traduire vos pages HTML. Très utile si vous avez une limitation au niveau du serveur que vous empèche de servir differentes pages pour chaque langue. dom-i18n é uma biblioteca em JavaScript que oferece uma alternativa para o suporte a internacionalização de páginas HTML. Bastante útil quando existem limitações no servidor que impedem a existência de diferentes versões da página para cada língua.
